# Fad lang

Language for solving algorithmic problems.

## Sample

```
Definition of numbers 0 .. 5
    N = 1, Z ++ = N, Z = 0
    N = 2, P ++ = N, P = 1
    N = 3, P ++ = N, P = 2
    N = 4, P ++ = N, P = 3
    N = 5, P ++ = N, P = 4

Definition of addition
    A + Z = 0, Z = 0
    A + B = C, A_2 + B = C_2, A ++ = A_2, C ++ = C_2

Find solution 2 + x = 5

    A + ? = B, A = 2, B = 5
```
